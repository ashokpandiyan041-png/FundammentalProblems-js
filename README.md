# JavaScript Fundamental Problems

This repository contains solutions to 10 beginner-friendly JavaScript problems designed to strengthen core programming fundamentals. The exercises focus on variables, conditional statements, loops, arrays, objects, and functions.

## Problems Covered

### 1. Even or Odd Number

Check whether a given number is even or odd.

### 2. Largest of Three Numbers

Find the largest number among three given numbers.

### 3. Sum of Numbers from 1 to N

Calculate the sum of all numbers from 1 to N using a loop.

### 4. Multiplication Table

Generate the multiplication table for a given number.

### 5. Positive, Negative, or Zero

Determine whether a number is positive, negative, or zero.

### 6. Access Object Values

Iterate through an object and display its keys and values.

### 7. Voting Eligibility

Check whether a person is eligible to vote based on age.

### 8. Array Basics

* Find the type of an array
* Find the length of an array
* Access array indexes and values

### 9. Smallest Number in an Array

Find the smallest value present in an array.

### 10. Reverse an Array Without Using reverse()

Print array elements in reverse order using loops.

Through these exercises, I practiced:

* JavaScript Fundamentals
